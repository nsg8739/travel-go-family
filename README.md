# 여행가자GO

시오니네 가족여행을 PC와 모바일에서 함께 관리하는 웹앱입니다.

- 배포 주소: https://nsg8739.github.io/travel-go-family/
- 저장소: https://github.com/nsg8739/travel-go-family

## 주요 기능

- 여러 여행 플랜 저장 및 지난 여행 조회
- 전체 일정·날짜별 일정 보기와 시간순 정렬
- 날짜별 Google 지도 동선 확인
- 일정 입력 경비와 여행 경비 자동 연동
- 경비 수정·삭제·순서 변경 및 예산 현황
- 여행 정보, 숙소 정보, 준비물 체크
- 숙소 체크인·체크아웃, 예약번호, 상세 링크 관리
- 예약번호가 일치하는 PDF 바우처 자동 연결
- 자주 쓰는 앱 바로가기와 Grab·Google 지도 실행
- 사칙연산 및 PHP↔KRW 환율 계산기
- JSON 데이터 백업·복원
- 관리자 이메일 로그인과 가족 읽기 전용 링크

## 관리자 데이터 동기화

Supabase 이메일·비밀번호 계정으로 로그인하면 같은 관리자 계정의 여행 데이터가 PC와 모바일에 동기화됩니다. 가족 초대 링크로 접속한 사용자는 읽기만 가능합니다.

Supabase를 새로 구성할 때 SQL 편집기에서 다음 파일을 순서대로 실행합니다.

1. `supabase-schema.sql`
2. `supabase-auth-complete.sql`
3. `supabase-delete-trip.sql`

## PDF 바우처

숙소 정보의 `PDF 자동 연결`에서 여러 PDF를 선택하면 파일명의 10자리 숫자와 숙소 예약번호가 같은 항목에 자동 연결됩니다.

PDF는 예약자 개인정보 보호를 위해 GitHub나 공개 링크에 올리지 않고 해당 기기의 IndexedDB에만 저장합니다. 따라서 다른 기기에서도 사용하려면 그 기기에서 PDF를 다시 연결해야 합니다.

## 환율 계산기

필리핀 페소(PHP)와 한국 원(KRW)을 양방향 환산합니다. Frankfurter API의 일일 환율을 사용하며 마지막 환율은 기기에 저장됩니다. 실제 환전소·카드 환율은 수수료에 따라 다를 수 있습니다.

## 로컬 실행

`index.html`을 직접 열어 기본 기능을 확인할 수 있습니다. 서비스 워커, 로그인, 외부 앱 실행 등 배포 환경 기능은 GitHub Pages 주소에서 확인하는 것이 정확합니다.

## 파일 구성

- `index.html`: 화면 구조
- `style.css`: 반응형 디자인
- `app.js`: 여행 관리와 동기화 기능
- `sw.js`: 오프라인 캐시와 최신 화면 갱신
- `manifest.webmanifest`: 홈 화면 설치 설정
- `src/img/`: 앱 바로가기 이미지
- `supabase-*.sql`: Supabase 데이터베이스 설정
